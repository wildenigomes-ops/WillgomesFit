Conexão Supabase
Passo a passo:
1. Hospede os dois arquivos (index.html + cliente.html) no Netlify Drop — arraste os arquivos e pronto, gratuito.
2. Copie a URL gerada (ex: https://meusite.netlify.app) e cole em URL do sistema abaixo.
3. Crie conta gratuita em supabase.com → novo projeto → Settings → API → copie Project URL e anon public key.
4. No Supabase, crie as tabelas: clientes e questionarios (ambas com colunas id text e dados jsonb).
